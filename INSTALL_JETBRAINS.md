# HOW TO INSTALL JETBRAINS PACKAGES BY SNAP

```
snap search goland
Name    Version   Publisher   Notes    Summary
goland  2020.3.1  jetbrains✓  classic  A Clever IDE to Go
```

```
sudo snap install goland --classic
```
